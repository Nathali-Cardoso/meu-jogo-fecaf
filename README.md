# 🎮 Meu Jogo - Fecaf

Projeto desenvolvido para a disciplina de **Game Development** no curso de Análise e Desenvolvimento de Sistemas da faculdade **FECAF**.

---

## 🧠 Conceito

O jogo tem como tema **"Alienígena Perdido"**.  
O objetivo do jogador é **coletar diamantes** enquanto explora o cenário e evita obstáculos.

---

## 🎮 Mecânicas

- Movimento lateral com as setas **← →**
- **Pulo** com a tecla **Espaço**
- **Colisão** com obstáculos (Game Over)
- Sistema de **coleta de objetos**

---

## 🧩 Estrutura

- Desenvolvido com **Unity**
- Linguagem utilizada: **C#**
- Estilo: **Jogo 2D de plataforma**

---

## 🛠️ Como rodar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/Nathali-Cardoso/meu-jogo-fecaf.git

2. Abra o projeto no Unity (versão recomendada: 2021.x ou 2022.x)

3. Execute a cena principal (caminho: Assets/Scenes/)
